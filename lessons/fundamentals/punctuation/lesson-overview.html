This lesson introduces punctuation.

  1. Some punctuation can be written with or without leading or trailing spaces using a different stroke. For example, “. ” with a space can be stroked TP-PL while “.” without spaces can be stroked P-P.

  2. . TP-PL => F-PL (full stop, spaced)

    . P-P (decimal point, unspaced)

  3. ... SKWR-RBGS => SY-SHUN (suspension point, then capital)

    ... HR-PS => L-PS (ellipsis, then lowercase)

  4. ? H-F (question mark, spaced)

    ! SKHRAPL => SKLAM (exclamation mark, spaced)

  5. ' A*E (opening single quote/apostrophe, no space)

    ' AE (closing single quote/apostrophe, space)

  6. " KW-GS => Q-SHUN (opening quotation mark, no space)

    " KR-GS => C-SHUN (closing quotation mark, space)

  7. , KW-BG (comma, spaced)

  8. ; SKWR*RBGS or STPH*FPLT (semi-colon, spaced)

    : STPH-FPLT (colon, spaced)

    : KHR-PB => KL-N (colon, unspaced)

  9. - H-PB => H-PB (hyphen), unspaced

    -- TK-RB => D-SH (dash)

    - PH*PBS => M-NS (minus, spaced)

  10. + PHR*US => PL*US (plus)

  11. = KWA*LS => QA*LS (unspaced equals)

    = KW-L => Q-L (spaced equals)

  12. * STA*R (star)

  13. < AEPBGT or AEPBG => ANGT (opening angle bracket)

    > A*EPBGT or A*EPBG => A*NGT (closing angle bracket)

    < HR*PB => L*N (less than)

    > TKPWR*PB => GR*N (greater than)

    < PWRABG => BRAK (opening angle bracket)

    > PWRA*BG => BRA*K (closing angle bracket)

  14. ( PREPB => PREN (opening parenthesis)

    ) PR*EPB => PR*EN (closing parenthesis)

  15. [ PWR-BGT => BR-KT (opening bracket)

    ] PWR*BGT => BR*KT (closing bracket)

  16. { TPR-BGT => FR-KT (opening French bracket)

    } TPR*BGT => FR*KT (closing French bracket)

  17. / OEU (forward slash, unspaced)

    \ SPWHRAERB => SBLAESH (backslash)

  18. ~ T*LD (tilde, spaced)

    ~ T*EULD (tilde, unspaced)

  19. @ KWRAT => YAT (at-sign)

  20. # HAERB => HAESH (hash)

  21. $ TK-PL => D-M (dollar mark)

  22. % PERS (percent)

  23. ^ KR-RT => C-RT (caret)

  24. & SP-PBD => SP-ND (ampersand)

  25. _ R*UPBD or RUPBD => RUND (underscore)

  26. | PAO*EUP (pipe)

  27. ` KH-FG => CH-VG (grave character, backwards)

    ` TR-RL (opening backquote, grave brief inverted)

    ` TR*RL (closing backquote, grave brief inverted)
